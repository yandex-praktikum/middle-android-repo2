# PraktikumChatApp

## Для ревьюров

В проекте есть 2 ветки: start, finish

- _start_ – ветка, в которой содержится начальный код. Здесь будут работать студенты.
- _finish_ – ветка, в с кодом решения всех заданий.

Текст заданий в вики https://wiki.yandex-team.ru/users/netimen/middle-android/project-module-2/
